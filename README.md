# Excel Translator v2.0

Улучшенный инструмент для перевода Excel файлов на русский язык с графическим интерфейсом.

## Возможности

- ✅ Графический интерфейс для выбора файлов
- ✅ Выбор конкретных листов и колонок для перевода
- ✅ Прогресс-бар и детальное логирование
- ✅ Повторные попытки при ошибках перевода
- ✅ Задержки между запросами для избежания блокировок
- ✅ Кэширование переводов для оптимизации
- ✅ Сохранение форматов данных

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```
